## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Execute in development

1. clone the repository
2. execute

```
yarn install
```
3. have NEST CLI already installed

```
npm i -g @nestjs/cli
```

4. run the database up

```
docker-compose up -d
```

## Stack Used
* MongoDB - Mongoose
* Nestjs
